# betterxeneon-spotify (deferred)

Standalone deep-Spotify widget — superseded by the unified
[`betterxeneon-media`](../betterxeneon-media/) widget, which combines:

- A general media player (works with anything publishing to Windows SMTC —
  Spotify, browser tabs, foobar2000, etc.)
- Spotify-specific bottom quadrants (Queue + Playlists) that show only when
  Spotify is the current SMTC source and the user is authenticated

This folder is kept as a marker; no code lives here. Spotify Web API integration
is a follow-up to the media widget — see `BetterXeneonWidget.Host.Spotify` (TBD).
